# CodeZen Test

Ce repository contient un projet test pour **CodeZen**, mon side business de **templates de sites web préfaits** destinés à être personnalisés et vendus à des clients.

---

## 🚀 À propos

**CodeZen** propose des templates modernes, minimalistes et **responsive mobile-first**, prêts à l’emploi pour des sites vitrines.  
Ce projet sert à montrer la structure et le workflow de mes templates.

---

## 💻 Technologies utilisées

- HTML5
- CSS3
- JavaScript (si nécessaire pour interactions simples)
- Déploiement sur **Netlify** pour une livraison rapide au client
